CSS 기본 선택자

선택자 란?
특정한 html 태그를 선택할 때 사용하는 기능으로, 태그를 선택하여 원하는 스타일과 기능을 적용할 수 있다

모든 선택자

html 안의 모든 태그를 선택할 때 전체 선택자를 사용할 수 있다.

  • 테스트2
  • 테스트3
